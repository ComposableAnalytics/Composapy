Environment Variables

If running Composapy in a DataLab, your environment variables will already be set up on your running instance. If running outside of a DataLab instance, you will need to set “DATALAB_DLL_DIR”. “APPLICATION_URI” can also optionally be set, however it is only used to create a Session object and can be passed through it’s “uri” argument.

DATALAB_DLL_DIR

The value of this should be a path to the required Composable dll’s needed to run Composapy. Composapy uses the PythonNet package to bind to the Composable project dll’s.

APPLICATION_URI

See Session.
